# Birney Bike Bus

A static one-page site for the Birney Bike Bus — a parent-led group ride to Alice Birney Elementary.

## Deploy to GitHub Pages

1. Create a new repository and upload both files (`index.html`, `social-card.png`) to the root.
2. In **Settings → Pages**, set **Source** to your default branch (`main`) and folder to `/ (root)`.
3. Your site publishes at `https://<user>.github.io/<repo>/`.

`index.html` is fully self-contained (inlined CSS, SVG, and the embedded map). `social-card.png` is only used for the social link preview (`og:image`).
